Get all Countries
URL: countries
Method: get
Return data: 
{{'[{ id: number, name: string },{...}]'}}
Get all states of a country
URL: states/{{ '{countryId}' }}
Method: get
Return data: 
{{'[{ id: number, name: string },{...}]'}}
Get Problems
URL: problems
Method: get
Query Paramters: 
  countryId: I.D of the country
  stateId: I.D of the state
  eco: (e, c or o) i.e Environment, Community or Organization
  ico: (i, c or o) i.e Individual, Community or Organization
  ecoUn: See me if you really really want it :)
  page: page number
  pageSize: number of items to send back
Return data: 
  {{'{ 
    totalItems: number, 
    items:[{problemId, ...}, {...}] 
  }'}}
Comments
URL: comments
Method: get
Query Paramters: 
    problemId: I.D of the problem
    page: page number
    pageSize: number of items to send back
Return data: 
    {{'{ 
      totalItems: number, 
      items:[{comments}, {...}] 
    }'}}
